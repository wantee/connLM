Get Started

Build the Toolkit

First, clone the project using git:

1 $ git clone https://github.com/wantee/connLM.git connLM

Then build the project from source:

1 $ cd connLM/src
2 $ ./autogen.sh
3 $ make -j 4

Running the examples

The example scripts are under egs/. egs/tiny is a tiny example for demonstration. Go to the directory and type run.sh will train several models one by one.

1 $ cd connLM/egs/tiny
2 $ ./run.sh

After this, you should see it is training models and report the final perplexity of test set.

There are other examples, some of them may need to download the training corpus from internet. You can run run.sh to play around.

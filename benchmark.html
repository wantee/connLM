Benchmark of the connLM toolkit

Experiment Environment

All experiments are run on the AWS c4.4xlarge instance , which has 16 vCPU @ 2.90GHz.

The binaries are compiled by gcc 4.8.3 with Intel MKL as the BLAS libary. Thanks to Intel, The MKL license is at no cost for open source projects.

All Detail Results can be found in egs/corpus-name/RESULTS.txt

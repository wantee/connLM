Update weights.

For WT_UT_FULL: in is [ col x 1 ]; er is [ 1 x row ]; For WT_UT_PART: in is [ col x 1 ]; er is [ 1 x row_seg.n ]; For WT_UT_SEG: in is [ col x 1 ]; er is [ 1 x updater->segs[row_seg_id].n ]; For WT_UT_ONE_SHOT: in is NULL; er is [ 1 x row ]; updating cols in in_idx of wt;

Parameters
[in]wt_updaterthe wt_updater.
[in]row_segsegment of wt row corresponding to error vector.
[in]row_seg_idid of segment of wt row, used by WT_UT_SEG.
[in]erthe error vector.
[in]er_scalescale of error vector.
[in]inthe input vector.
[in]in_scalescale of input vector.
[in]in_idxinput index (with scale) of input one-shot vector.
Returns
non-zero value if any error.
